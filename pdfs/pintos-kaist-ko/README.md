# Pintos KAIST 한국어 로컬 문서

원본: https://casys-kaist.github.io/pintos-kaist/

생성된 문서 수: 33개 + 학습 가이드

## 보는 방법

브라우저에서 `index.html`을 열면 됩니다.

```bash
open docs/pintos-kaist-ko/index.html
```

또는 정적 서버로 볼 수 있습니다.

```bash
python3 -m http.server 8080 -d docs/pintos-kaist-ko
```

그 다음 `http://localhost:8080`을 열면 됩니다.
