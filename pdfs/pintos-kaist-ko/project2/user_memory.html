원문 보기
KAIST Pintos GitBook 원문을 한국어로 옮긴 로컬 읽기본입니다. 코드, 명령어, 함수명, 파일명은 의도적으로 원문 형태를 최대한 유지했습니다. 의미가 애매한 부분은 원문 링크와 함께 확인하세요.

사용자 메모리 접근

사용자 메모리 접근을 구현합니다.

시스템 콜을 구현하려면 사용자 가상 주소 공간의 데이터를 읽고 쓸 수 있는 방법을 제공해야 합니다. 인자를 가져오는 단계에서는 이 기능이 필요하지 않습니다. 하지만 시스템 콜 인자로 전달된 포인터에서 데이터를 읽을 때는 이 기능을 통해 접근해야 합니다. 이 부분은 조금 까다로울 수 있습니다. 사용자가 잘못된 포인터를 넘기면 어떻게 해야 할까요? 커널 메모리를 가리키는 포인터를 넘기거나, 접근하려는 블록 일부가 그런 영역에 걸쳐 있으면 어떻게 해야 할까요? 이런 경우에는 사용자 프로세스를 종료하는 방식으로 처리해야 합니다.
